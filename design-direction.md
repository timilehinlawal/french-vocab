# French Vocabulary Trainer Design Direction

## North Star

The product should feel like a calm, modern learning app: simple, spacious, mobile-first, and quietly premium. The closest references are images 3-5: soft app surfaces, confident typography, clear cards, bottom navigation on mobile, and very little visual noise.

## Principles

- Use a neutral canvas with black/graphite type and subtle warm-gray surfaces.
- Typography should feel clean and phone-native: Avenir-style sans, calm sizing, no negative tracking, and fixed breakpoint sizes rather than viewport-scaled type.
- Surfaces should feel soft and translucent, closer to frosted task cards than heavy dashboard panels.
- Keep actions obvious and icon-led, with compact text only where it improves clarity.
- Prefer list-like learning surfaces over dense dashboard blocks.
- Use rounded app controls and soft glassy panels, but keep content cards restrained and structured.
- Let one strong typographic moment carry each page; avoid competing headings.
- Keep vocabulary tools separate and readable: synonyms, word family, and usage patterns should never run together.
- Use accents sparingly: thin gradient lines, soft photographic-feeling surfaces, and quiet depth.
- Mobile should feel native: bottom navigation, thumb-friendly controls, and no horizontal overflow.

## Page Direction

- Home: a simple personal status page with one large learning state, a compact today summary, and a clear next action.
- Vocabulary: a clean knowledge-base list with level metadata, meaning, example, status, and structured learning tools.
- Review: a focused flashcard scene with the rating controls available near the thumb on mobile.
- Analytics: only essential progress signals, shown as calm tracking rather than heavy reporting.
